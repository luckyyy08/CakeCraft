🎂 CakeCraft – Premium Online Bakery & Custom Cake Shop
🌐 Project Overview
CakeCraft is a sophisticated Full-Stack Web Application developed as a flagship project for the TYBSc Computer Science curriculum. It serves as a digital gateway for cake enthusiasts to explore, customize, and book artisanal cakes for their special celebrations. This project stands as a testament to modern web development practices, focusing on seamless user experiences and robust backend management.

📝 Project Essence
CakeCraft is engineered to bridge the gap between local bakeries and digital consumers. It features a secure environment where users can manage their personal profiles, track orders, and curate their favorite sweet treats. The core objective was to implement high-level State Management using PHP Sessions and create a fluid, Single-Page Application (SPA) feel within a multi-page architecture.

💻 Technical Excellence (Tech Stack)
Frontend Craftsmanship: HTML5, CSS3, JavaScript (ES6+)

UI/UX Frameworks: Bootstrap 5, Font Awesome Icons

Backend Architecture: PHP (Procedural & Secure Logic)

Database Management: MySQL (Relational Database)

Development Environment: XAMPP (Apache Server)

Version Control: Git & GitHub

🚀 Elite Features
Secure Authentication Shield: Encrypted user registration and login protocols.

Smart Navigation Logic: Intelligent redirection that detects session status to protect private routes.

Dynamic User Dashboard: A centralized hub to manage Orders, Addresses, and Settings without page reloads.

Interactive Slide Panel: A sleek, JavaScript-driven overlay for real-time profile editing.

Responsive Fluidity: A "Mobile-First" design approach ensuring pixel-perfect display across all devices.

🧠 Learning Milestones
Mastering Session-based Security to safeguard user data.

Implementing CRUD operations (Create, Read, Update, Delete) via PHP and MySQL.

Optimizing UI performance through DOM Manipulation and Asynchronous Logic.

Adopting professional Code Documentation and Repository Management standards.
